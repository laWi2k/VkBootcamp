# Тестовое задание на VKBootcamp в Сириусе

Приложение, которое отображает список сервисов VK. Сделано кодом с помощью UIKit, SafariServices и Reachability. Реализована проверка на подключение к сети.
Адаптируется под тему девайса (темная, светлая). Асинхронная загрузка иконок приложений.
